Roth Conversion

The Roth Conversion widget allows users to explore whether a Roth IRA conversion would be beneficial for their financial situation.

This widget can be found on the NewRetirement website at https://www.newretirement.com/retirement/roth-conversion-calculator/.

tip

You can now use the year attribute to select calculations and data for a given tax year; currently the widget supports data for 2022 and 2023. To ensure your widget displays values for 2023, make sure you set year to 2023 as shown below.

Embed Code
<script type="module" src="https://widgets.newretirement.com/roth-conversion/index.js"></script>
<newretirement-roth-conversion year="2023"/>
