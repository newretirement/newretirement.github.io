Getting started

Create a plan on NewRetirement.com​

If you haven’t used the NewRetirement.com Planner yet, we recommend creating a free account and creating a financial plan. The free tier will give you a great overview of the platform’s underlying approach and capabilities, including:

  • the key concepts and terms we use to describe lifelong financial planning and wellness;
  • the essential inputs that help to build accurate planning scenarios;
  • and the comprehensive range of options we support for users to add greater nuance and detail to their financial plans.

Get an API key​

Next, you’ll need an API key to access the NewRetirement Financial Planning Engine — even if you’re just interested in embedding a widget on your website.

If you’re simply kicking the tires, you can use the demo key below. For any other use, you’ll want your own API key; contact customer_service@newretirement.com to get your own key.

Next steps​

If you’re interested in adding a web component to your application, look to the widget catalog for information about embed codes, theming, and more.

If you’re interested in deeper levels of integration, we recommend going to Concepts and models for more detailed information about integrating your application with the FPE and our SDKs.
