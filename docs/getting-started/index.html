Getting started

Create a plan on NewRetirement.com​

If you haven’t used the NewRetirement.com Planner yet, we recommend creating a free account and creating a financial plan. The free tier will give you a great overview of the platform’s underlying approach and capabilities, including:

  • the key concepts and terms we use to describe lifelong financial planning and wellness;
  • the essential inputs that help to build accurate planning scenarios;
  • and the comprehensive range of options we support for users to add greater nuance and detail to their financial plans.

Get an API key​

Next, you’ll need an API key to access the NewRetirement Financial Planning Engine — even if you’re just interested in embedding a widget on your website.

If you’re simply kicking the tires, you can use the demo key below. For any other use, you’ll want your own API key; contact sales@newretirement.com to get your own key.

caution

Our demo key is designed for exploratory use of the FPE API only. Even though it provides access to all endpoints, requests are throttled — and of course, you’ll be sharing it with other demo users.

def send_request():

import org.apache.http.client.fluent.*;

public class SendRequest {
    public static void main(String[] args) {
        sendRequest();
    }

    private static void sendRequest() {
        // Request (GET )
        try {
            // Create request
            Content content = Request.Get("https://api.newretirement.com/fpe/v5/info")
                    // Add headers
                    .addHeader("Accept", "application/json")
                    .addHeader("x-api-key", "VDlPH9oTUp8NdzwzDMLpv5h0knU6fZft5YNxtb5Q")
                    // Fetch request and return content
                    .execute().returnContent();

            // Print content
            System.out.println(content);
        }
        catch (IOException e) {
            System.out.println(e);
        }
    }
}
