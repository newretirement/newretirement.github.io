Welcome

Thanks for checking out the NewRetirement development documentation!

The NewRetirement Platform is a suite of web components, SDKs, and APIs that enable you to create unique financial planning applications leveraging the same components that have been used to build the NewRetirement.com Planner.

There are a number of different approaches for integrating NewRetirement technology into your application or website, depending on your requirements.

  • Widgets — Web components that can be embedded in your web page or application
  • APIs — Use NewRetirement’s Financial Planning Engine to create and present financial simulations to your application users

For enterprise customers interested in even closer integration, the NewRetirement Planner is also available as a hosted, white-label application. For more information, contact sales@newretirement.com.
