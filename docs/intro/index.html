Welcome

Thanks for checking out the Boldin development documentation!

The Boldin Platform is a suite of web components, SDKs, and APIs that enable you to create unique financial planning applications leveraging the same components that have been used to build the Boldin.com Planner.

There are a number of different approaches for integrating Boldin technology into your application or website, depending on your requirements.

  • Widgets — Web components that can be embedded in your web page or application
  • APIs — Use Boldin’s Financial Planning Engine to create and present financial simulations to your application users

For enterprise customers interested in even closer integration, the Boldin Planner is also available as a hosted, white-label application. For more information, contact sales@boldin.com.
